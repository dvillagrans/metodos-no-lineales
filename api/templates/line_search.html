Métodos de optimización unidimensional

Los métodos de búsqueda de línea resuelven el subproblema unidimensional:

$$\min_{\alpha > 0} f(x_k + \alpha d_k)$$

Donde $x_k$ es el punto actual, $d_k$ es la dirección de búsqueda, y $\alpha$ es el tamaño del paso.

Ejemplo Numérico
Para $f(x) = x^2$, $x_0 = 2$, $d = -1$:
$\phi(\alpha) = (2 - \alpha)^2$
Mínimo en $\alpha^* = 2$
Sección Áurea

Utiliza la proporción áurea $\phi = \frac{1+\sqrt{5}}{2} \approx 1.618$

$\tau = 0.618$
